## About
This folder has all the basic text files for different versions of Xenophon's *Anabasis*.
* English Translation: Carleton Lewis Brownson (Loeb Classical Library, London 1922), retrieved from ToposTexts (https://topostext.org/work/90).
* Ancient Greek: Retrieved from the Perseus Digital Library (https://www.perseus.tufts.edu/hopper/text?doc=Perseus:text:1999.01.0201).
