# Tracking-the-Ten-Thousand
### This repository houses data from Tracking the Ten Thousand, an ongoing research project by Cian Colgan. Tracking the Ten Thousand analyzes, explores, and reconstructs the Anabasis of Xenophon, an Ancient travelogue, using semantic geo-annotation and GIS modeling.
Here you can find the various plain text files of the text, geo-annotation data from those texts, and the many GIS files used for analysis of the text and reconstruction of the traveled route.

For more documentation on the project or information on me, please see: **https://ciancolgan.com/**
