** About
This folder contains all GIS related files for the geometry, associated metadata, etc. Analysis was done using ArcGIS Pro (v. 3.3.0). Files can be imported and used in other GIS softwares, like QGIS.

Additional data:
* For basemap used: CAWM Map Tiles (https://cawm.lib.uiowa.edu/index.html)
* For additional ancient spatial data: AWMC repo (https://github.com/AWMC/geodata)
* Reccomended DEM sources: NASA's SRTM-GL1 (for data, https://portal.opentopography.org/raster?opentopoID=OTSRTM.082015.4326.1) or USGS's GMTED2010 (for data, https://www.usgs.gov/coastal-changes-and-impacts/gmted2010)
